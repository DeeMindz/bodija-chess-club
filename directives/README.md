# Directories

This directory contains the directive files that define tasks for the AI agent.

## Purpose
Directives are natural language instructions that define:
- Goals: What needs to be accomplished
- Inputs: What data or parameters are needed
- Tools/Scripts: Which execution scripts to use
- Outputs: Expected results
- Edge Cases: Potential issues and how to handle them

## Creating New Directives
When you need to automate a new task:
1. Create a new `.md` file in this directory
2. Define the goal, inputs, tools, outputs, and edge cases
3. The Orchestrator (Layer 2) will read this directive and execute accordingly
